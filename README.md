# 🏡 Real Estate Market Analysis

## 📖 About the Project

This project is a Python-based **Exploratory Data Analysis (EDA)** on a real estate dataset. It cleans and preprocesses the data, then analyzes property prices, localities, builders, and other market trends to generate meaningful business insights. The project uses **Pandas** for data manipulation and **Matplotlib** & **Seaborn** for data visualization. 

## 🚀 Features

- Data Cleaning & Preprocessing
- Price and Rate per Sq. Ft. Analysis
- Locality-wise Price Comparison
- RERA Approval Analysis
- Ready-to-Move vs Under Construction Comparison
- BHK & Flat Type Analysis
- Builder-wise Analysis
- Data Visualization

## 🛠️ Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn

## ▶️ Run the Project

```bash
pip install pandas matplotlib seaborn
python main.py
```

## 📊 Key Insights

- Costliest property
- Highest-priced locality
- Highest rate per square foot
- Impact of RERA approval on price
- Relationship between area and price
- Most expensive BHK & flat type
- Top premium builders
